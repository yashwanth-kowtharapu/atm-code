# ATM Simulator

## Overview
The **ATM Simulator** is a Java-based console application that simulates the basic operations of an Automated Teller Machine (ATM).  
It allows users to securely perform banking operations such as checking balances, depositing money, withdrawing funds, and viewing transaction history.

This project is intended as a learning tool for Java programming, object-oriented design, and simple banking logic.

---

## Features
- **User Authentication**: Login using a username and password.
- **Multiple Accounts Support**: Simulate multiple bank accounts.
- **Check Balance**: View the current balance of your account.
- **Deposit Money**: Add funds to your account.
- **Withdraw Money**: Withdraw funds with balance validation.
- **Transaction History**: Keep a simple record of deposits and withdrawals.
- **Console-based Interface**: Simple menu-driven interface for ease of use.
